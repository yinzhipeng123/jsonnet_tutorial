语法


注释：


三种注释
[root@centos7 jsonnet_learn]# cat test.jsonnet 
/* 注释1 */
#  注释2
// 注释3
{
served: 'Tall',
agent:"Tom",
}
[root@centos7 jsonnet_learn]# jsonnet test.jsonnet 
{
   "agent": "Tom",
   "served": "Tall"
}




字符串：


单行字符串用双引号

"

或者单引号

'

引起来。
[root@centos7 jsonnet_learn]# cat test.jsonnet 
{
served: 'Tall',
agent:"Tom",
}
[root@centos7 jsonnet_learn]# jsonnet test.jsonnet 
{
   "agent": "Tom",
   "served": "Tall"
}

如果字符串中出现单引号，单引号前加\进行转义。例如"Farmer's Gin" --> 'Farmer\'s Gin'
[root@centos7 jsonnet_learn]# cat test.jsonnet 
{
served: 'Farmer\'s Gin'
}
[root@centos7 jsonnet_learn]# jsonnet test.jsonnet 
{
   "served": "Farmer's Gin"
}

允许跨多行字符串可以用 ||| 进行标记，例如

[root@centos7 jsonnet_learn]# cat test.jsonnet 
{
description: |||
        The Tom Collins is essentially gin and
        lemonade.  The bitters add complexity.
      |||,}
[root@centos7 jsonnet_learn]# jsonnet test.jsonnet 
{
   "description": "The Tom Collins is essentially gin and\nlemonade.  The bitters add complexity.\n"
}
      


对象和数组可以尾随，



正常的一个json例子

{
"Fruits": [
"apple",
"Banana"
]
}
如果加几个逗号，那么该json为错误了

对象或者数组末尾的一个不能有逗号，但是jsonnet却可以正确的识别
[root@centos7 jsonnet_learn]# cat test.jsonnet 
{
        "Fruits": ["apple", "Banana",],
}

[root@centos7 jsonnet_learn]# jsonnet test.jsonnet 
{
   "Fruits": [
      "apple",
      "Banana"
   ]
}


KEY值可以不用加引号"



[root@centos7 jsonnet_learn]# cat test.jsonnet 
{
        Fruits: ["apple", "Banana",],
}
[root@centos7 jsonnet_learn]# jsonnet test.jsonnet 
{
   "Fruits": [
      "apple",
      "Banana"
   ]
}
