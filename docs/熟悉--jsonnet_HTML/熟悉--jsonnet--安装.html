安装




github项目地址
https://github.com/google/jsonnet


环境：CentOS7

使用C版本的jsonnet，不要使用GO或者Python提供的，不知道为什么GO或者Python去跑测试的时候，都不通过

#cd /usr/local/src/

#git clone -b v0.17.0 https://github.com/google/jsonnet.git

#cd jsonnet

#yum install -y gcc gcc-c++

#make

#vim /etc/profile
末尾添加
export JSONNET_HOME=/usr/local/src/jsonnet
export PATH=$JSONNET_HOME:$PATH
